# Customer Shopping Behavior Analysis

## Overview

This project analyzes customer shopping behavior to understand purchasing patterns, product preferences, and revenue trends. The analysis helps identify key insights that support data-driven decision making for businesses.

## Dataset

The dataset contains customer shopping records including customer ID, gender, product category, item purchased, purchase amount, review rating, and subscription status. It includes around **3,900 customer transactions** used to analyze buying patterns and customer segments.

## Tools Used

* Python (Pandas, NumPy)
* MySQL
* Power BI
* Jupyter Notebook
* Microsoft PowerPoint

## Project Steps

1. Loaded the dataset using Python in Jupyter Notebook.
2. Performed **Exploratory Data Analysis (EDA)** to understand data patterns.
3. Cleaned and prepared the dataset for analysis.
4. Stored the processed dataset in **MySQL**.
5. Performed business analysis using **SQL queries**.
6. Built an **interactive dashboard in Power BI**.
7. Created a **project report and presentation**.

## Dashboard

The Power BI dashboard visualizes important insights such as:

* Revenue by gender
* Top rated products
* Most purchased products by category
* Customer purchase behavior patterns

## Results / Insights

* Identified product categories with the highest sales.
* Analyzed customer purchase behavior based on gender and subscription status.
* Found top-rated products and popular product categories.
* Provided visual insights to support business decisions.

## How to Run the Project

1. Open the **Jupyter Notebook** file to view data cleaning and analysis.
2. Run the **SQL queries** in MySQL to perform business analysis.
3. Open the **Power BI (.pbix)** file to explore the dashboard.
4. Refer to the **presentation and report** for a summary of insights.
